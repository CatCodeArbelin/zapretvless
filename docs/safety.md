# Safety boundaries

PR-01 bootstrap намеренно ограничен безопасной инфраструктурной заготовкой. В текущих границах проекта явно отсутствуют:

- VLESS parser;
- Xray/Zapret engines;
- запуск внешних binaries;
- изменения DNS, proxy или routes;
- installer, telemetry или auto-update.

Эти ограничения фиксируют, что bootstrap не выполняет сетевую настройку системы, не запускает сторонние движки и не содержит механизмов установки, телеметрии или автоматического обновления.
