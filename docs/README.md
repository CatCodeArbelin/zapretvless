# Документация

Краткий обзор PR-01 bootstrap:

- репозиторий содержит минимальный .NET bootstrap для решения `zapretvless.sln`;
- в состав входят проекты клиента, сервиса, общей библиотеки и тестов;
- проверка состояния bootstrap выполняется через стандартные команды `dotnet` и скрипт `scripts/dev-check.ps1`;
- Docker Compose на этом этапе используется только для проверки конфигурации, без запуска рабочих сетевых компонентов.

Подробности для разработчика см. в [`dev-setup.md`](dev-setup.md), а границы безопасности — в [`safety.md`](safety.md).
